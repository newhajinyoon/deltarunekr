자주 묻는 질문

더빙은 어떻게 켜고 끄나요?

세이브 파일을 선택하는 화면에서 "한더빙"을 눌러 끄거나 키시면 됩니다.
(노란색인 상태가 켜진 상태입니다.)

델타룬 비공식 한글 패치는 상업적인 프로젝트인가요?

아닙니다. 이 프로젝트는 비영리로 진행되며, 어떠한 수익도 창출하지 않습니다.

패치에 포함된 언어 파일을 수정해도 되나요?

네, 개인적인 용도로 수정하는 것은 자유입니다. 다만, 수정된 파일을 배포할 계획이 있다면 반드시 출처를 명시하고, 패치 팀의 동의를 받아 주세요.

패치가 적용된 게임이나 프로젝트 자체를 유료로 판매하면 어떻게 되나요?

해당 행위는 명백한 금지사항입니다. 이런 사례가 발견될 경우 즉시 신고해 주세요.
