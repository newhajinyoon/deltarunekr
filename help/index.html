FAQ

마지막 업데이트 2025년 7월 29일

자주 묻는 질문

더빙은 어떻게 켜고 끄나요?

세이브 파일을 선택하는 화면에서 "한더빙"을 눌러 끄거나 키시면 됩니다.
(노란색인 상태가 켜진 상태입니다.)

델타룬 비공식 한글 패치는 상업적인 프로젝트인가요?

아닙니다. 이 프로젝트는 비영리로 진행되며, 어떠한 수익도 창출하지 않습니다.

패치에 포함된 언어 파일을 수정해도 되나요?

네, 개인적인 용도로 수정하는 것은 자유입니다. 다만, 수정된 파일을 배포할 계획이 있다면 반드시 출처를 명시하고, 패치 팀의 동의를 받아 주세요.

패치가 적용된 게임이나 프로젝트 자체를 유료로 판매하면 어떻게 되나요?

해당 행위는 명백한 금지사항입니다. 이런 사례가 발견될 경우 즉시 신고해 주세요.

설치 파일이 바이러스로 감지되는데 안전한가요?

네, 바이러스나 악성 코드는 전혀 포함되어 있지 않습니다. 보안 프로그램이 인증되지 않은 파일이라고 오탐할 수 있지만, 안심하고 설치하셔도 됩니다.
(만약 걱정되신다면 사이트 메인페이지 하단에서 수동 패치 방식도 제공하고 있으니 참고해 주세요.)

패치에 사용된 에셋을 모딩이나 다른 용도로 활용해도 되나요?

네, 모딩 및 다른 모드들과의 한글 패치 호환성을 목적으로 한글 패치에 사용된 일부 에셋들을 이곳(구글 드라이브)을 통해 공유하고 있습니다.

다만 상당수의 요소들에 델타룬의 원본 에셋 코드와 이미지가 남아있으므로, 재배포는 엄격히 금지되며 모든 권리는 델타룬 팀에 귀속됩니다.

macOS 환경에서도 한글 패치를 사용할 수 있나요?

현재 dtkrpatchteam은 비영리로 활동하고 있으며, macOS를 지원하는 기기로 패치를 만드는 것에 어려움이 있습니다. 추후 macOS 지원 기기로 패치 프로그램이 정상 작동하는 것을 확인한 후 업데이트가 있을 예정이니 양해 부탁드립니다.

챕터 5가 출시되면 한글 패치는 언제 나오나요?

챕터 5의 한글 패치 또한 챕터가 나오는 즉시 개발에 착수할 예정입니다. 다만 외부 요소에 따라 일정에 변수가 있을 수 있음을 미리 알려드립니다.
